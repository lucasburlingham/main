# Uptime Counter

## Description

Measures the uptime of your internet connection and displays it in a simple way.

## Installation

### Requirements

- Python 3.6+
- pip
- make
- virtualenv
- sqlite3

### Install

```bash
make install
```

## Usage

```bash
make run
```

or

```bash
flask run
```
